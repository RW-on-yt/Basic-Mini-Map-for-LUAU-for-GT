# Basic-Mini-Map-for-LUAU-for-GT
As the name suggest this is a unity package for gorilla tag custom maps.
## Contents
  - Basic Non-Networked System for the Mini Maps
  - Basic Networked System for the Mini Maps

# How to use
  - Follow the main steps in the GTAG custom map mod.io guide to create your map
  - Drag in this unity package like any other
  - Now set the script that will be placed in the "Mini Map" as the LUAU script
  - export your map and see the cool little monkeys
